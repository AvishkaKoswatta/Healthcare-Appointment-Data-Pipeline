Lightweight demo pipeline for streaming appointment events from PostgreSQL -> Kafka (Debezium) -> S3 -> Snowflake with Airflow orchestration and a Flask dashboard. 

Key components  
--------------
1. Docker 
2. Postgresql
3. Kafka + Debezium connector 
4. Amazon S3
5. Snowflake
6. Airflow & dbt
7. Web dashboard
